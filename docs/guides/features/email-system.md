# Email System

The application sends automated emails for various purposes including authentication, notifications, and updates.

## Email Types

### Authentication
- **OTP Codes**: One-time passwords for login
- **Welcome Messages**: Account setup confirmations

### Applications
- **Submission Confirmations**: When applications are submitted
- **Status Updates**: When application status changes
- **Interview Invitations**: Scheduling information

### Administrative
- **New Application Alerts**: Notifications for admins
- **System Updates**: Important announcements

## Features

- **Reliable Delivery**: Multiple provider support
- **Professional Templates**: Consistent branding
- **Secure Content**: Safe HTML and link handling
- **Delivery Tracking**: Monitor email success
- **Error Handling**: Automatic retries and fallbacks

## Configuration

Emails are configured through environment variables for SMTP provider settings and delivery options.
