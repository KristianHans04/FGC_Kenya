# Dashboard

The application provides different dashboard views depending on your role in the team.

## Student Dashboard

For team applicants, the dashboard helps you:
- Create and edit your application
- Upload supporting documents
- Track your application status
- View notifications and updates

## Admin Dashboard

For team administrators, the dashboard includes:
- Review submitted applications
- Manage user accounts
- View application statistics
- Send bulk communications
- Configure system settings

## Mentor Dashboard

For team mentors, you can:
- Review assigned applications
- Provide feedback to applicants
- Track student progress
- Schedule mentoring sessions

## Alumni Dashboard

For team alumni, the dashboard offers:
- Update your profile and achievements
- Connect with current team members
- Participate in alumni events
- Share success stories

## Features

- **Responsive Design**: Works on all devices
- **Real-time Updates**: See changes immediately
- **Secure Access**: Role-based permissions
- **Easy Navigation**: Intuitive menu system
